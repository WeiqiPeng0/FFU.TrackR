# FFUTrackR
On-going project (Croker Lab)

USE:
1. Install Rstudio
2. Install dependent packages
3. run "runApp()" in the directory

OR check out the online version at https://croker.shinyapps.io/ffutrackr/

 - server.R: the backend file
